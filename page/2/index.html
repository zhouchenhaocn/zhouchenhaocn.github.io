  • 饿

    2014-10-11 |
    • { 人文 }
    • { 随笔 }

      在这里写文章是幸福的，饿也是幸福的。

      饿是幸福的吗？不幸是饿到虚弱的时候身边还没有食物，但饿本身是幸福的，它让人们有了对食物的追求，对一切的追求，一切原动力都在此。当人们满足地把食物一口一口从嘴里送下肚的时候，幸福感是加倍的。饿让人们有了十足的存在感，饿是充实，是健康的证明。对饿的把握与控制是一种修行。饿让人们聚到了一起，让人们回家，让人们相互有爱。

      周末临近中午的时候出去买菜，出发的时候脚步是闲散的，回家的时候脚步是急急的，这也是因为饿。

      在菜场，尽管菜的位置不会变，我还是习惯先兜一圈看看。这是因为菜的品质和价格每次都有改变，比如番茄，两三块到六七块的单价之间是经常变动的。乘便宜的时候多买一点，然后等到下一个间隔再买是一般的做法。不过，便宜的时候也一定要注意，番茄的手感决定一切，往往太硬太黄的番茄，即使是促销最低价我也不会买。反之，遇到难得一见的品质番茄，即使贵一点，如果今天真的太想吃的话，我也会小心翼翼地拿两三个回去。

    2014-10-03 |
    • { 人文 }
    • { 随笔 }
    • { 音乐 }

      传说七弦琴在精通的人手中能够惊天地泣鬼神，各种音调转折能使星辰移动，风雷轰鸣，雨雪自积聚来的云中飞降大地。这真的是很浪漫。其实七弦琴本身并没有十足的力量，一张小小的琴，传出低调的琴声，时刻准备淹没在万物与世界之中。要说琴的奇妙力量，其实是心的力量。天地鬼神，雨雪星辰，大概只是在内心翻涌流动，但是，这就足够伟大了。

      我还是愿意保留对传说中精通古法的弹奏者的憧憬。那样的演奏，竟能使鬼神为之倾听而感动，更和自然的力量产生共鸣，好像世界和心之间真的相连一样。弹琴在此刻并非只是自我的事，而是倾注着弹奏者庄严和广阔的心，并非只是为谁弹奏，而是充盈着一种如斋戒和沐浴一般的对这个世界的崇敬。

  • 寄宿家庭的记忆

    2014-08-03 |
    • { 人文 }
    • { 随笔 }

      今天出门，校区旁Monash College里陆陆续续出来了很多刚到这里的学生，推着机场下来还未开封的行李，每个人的身旁都有一位寄宿家庭的家长陪伴。家长们手里拿着学生的资料，显然是刚和学生碰面的。学生们从机场直接被送到这里，和等待他们的家庭欣喜见面，然后一起回家。

  • 最好的时候

    2014-06-20 |
    • { 人文 }
    • { 随笔 }

      从五月到六月，由秋转冬，天气时不时要冷一下。室外草场的跑道上，冬天的风和阳光混合着。跑上半个小时的步，身体所感受到的季节和自然的信息就足以超过平常忙碌时候的累计了。呆在室内的时候，自己当然会有很多事情做，很多问题想，更有一些烦恼会来郁结，这些东西的重要与否自己会在不同的时段给出不同的答案，任何一种答案都可以算是聪明的回答，但只有在户外的时候，心里的感觉才最为真实可靠，内心和身体，身体与世界，在最常见的常态中，才会发觉很多东西的无关紧要。世界的气息变得柔和了，自己似乎可以感受未来的变化，百年之后，或者体会千年之前的气息，透过书本隐约窥见的，在户外便能近乎真切地触摸到。

      苍劲的大树一直在那里，我以自己年轻的最好的身体与它相遇在这个时候。我感到愉悦，也感到不安，但这种不安让我获得了持久的平静。身和心之间连得太紧了，心灵和思想从来不应该，也不会脱离身体而存在。我并不知道死亡之后会有怎样的变化，我也暂时不需要某种明确的答案，只是活着的时候，我想让自己的心灵踏踏实实地陪伴自己的身体，让自己的身体在需要滋润的时候得到滋润，在成长的时候得到成长，像父母守护孩子一样，我也用心去爱惜和珍视身体，用心去见证。
    用自己用心经营的生活方式滋润着身体，身体收获了愉悦，从而心灵也收获了愉悦，这或许是最上等的愉悦。

  • 跑步

    2014-04-24 |
    • { 自然 }
    • { 人文 }
    • { 随笔 }

      我喜欢上了家附近reserve的跑道，从第一次跑步到现在，脚下和眼前都熟悉起来，踩在路上，眼望前方，全身舒展开来，非常舒服。习惯了行进中有节奏的呼吸，听着脚下稳稳地声音，感觉可以一直跑下去。

      有风迎面持续吹来的时候，有阳光倾斜地照耀的时候，跑步变得更加享受。清爽的空气，傍晚的草坪，很多小小的鸟儿贴着草尖飞来飞去，近处观察，鸟儿油亮的白色腹部都看得一清二楚，滑翔而过，像在绿色的云朵上自由穿梭一样，想到这里，眼前的景色变得奇妙起来。

      但是，脚踩的大地总是比高处的云朵更加令人踏实和亲近，鸟儿们即使是一直在天空，大地想必还是它们的归宿。

      人们相逢相离，其中有断不了的想念和情愫，有些人彼此在向往着对方和对方的土地，有些人此时相隔千里。当这样的时空无论怎样都穿越不透的时候，那就怀着一颗宁静的心在大地上奔跑吧，大地藏着彼此的根，那是它的秘密。

  • 读平家物语

    2014-02-28 |
    • { 人文 }
    • { 随笔 }

      上午读完了《平家物语》，觉得精彩壮阔不能用言语表达。

      能通过这样绝妙的文字目睹八百多年前的世界以及其中的人物，我觉得三生有幸。这又是一本让我全身心融入其中的好书。如果说《源氏物语》的译者功底让我望尘莫及的话，这本《平家物语》的译者功底便能让我“绝望”。总之，细细读完两本书，虽然感到读的并不是原本，但是心里已经没有遗憾。

      两本书中涉及到很多佛教思想，其中《平家物语》里涉及的尤甚。在读这本书之前我恰巧在了解佛教的思想，之后再去读这本书，让我的思维更加活跃了，内心也更能追寻和感受八百年前的乱世和其中的人物，这样的衔接是一种很棒的缘分。

      译者在前言中曾说，那个时代佛教给人的影响，让他们在言行举止和面对生死命运的时候，都增加了一种温情。小说中也因为这些与佛有关的描写，无论是对故事的描写还是对人物心理的描写，在无限的悲苦之中让人多了一种“若有若无的温暖”，也让文学的感动“得到了最大限度的升华”。

      这是一直存着的一个题目，想要认真来写。前几天晚上写了一些，后来被我全部舍弃，觉得晚上在房间里开灯拉着窗帘，虽然很静，但是太过静了，五官与内心反而枯竭起来。房屋虽已经能完美地遮风避雨，但是四周又完全阻隔，关上窗便不透一点风声，打开灯便不觉一点夜色，这是有些遗憾的。

    2013-12-14 |
    • { 自然 }
    • { 随笔 }

      记录一下今天的好天气。

      入夏以来经历过暴升的高温，随后是连续的大风大雨，到最近天气才渐渐缓和。风清日宜，蓝天白云，空气中混合了花和草的味道，让人忍不住留恋在外。

      往常的高温天，白天是恼人的蝉声，晚上是遍地的虫鸣，温度越高，叫声越是团结肆意。在我的印象里，虫声就是为这样的夏天而鸣的。久居在城市的楼房里，在背景的喧闹中唯有如此嘹亮而持久的虫声才能被我注意到。这样的注意尚且是被动的，更不要说其他细腻的虫声了。书中的秋虫对我来说是个陌生的词汇，它们的鸣叫在特殊的季节里如摇铃般的悦耳，然而我全不曾注意，甚至怀疑起秋虫的存在了。

  • 琴曲笔录

    2013-09-08 |
    • { 人文 }
    • { 随笔 }
    • { 音乐 }

      学琴的时间跨度快一年了，老师那里的笔记是去年的9月16号。时间越长，琴缘越深，不论是和老师，还是和我所听到学到的琴曲。

      有人说，听西方古典要知道作者的故事，听古琴要知道琴曲的故事。每一首琴曲都有一个名字，名字包含的或是一个背后的故事，或是一个可以抒发的意象。听古琴，可以和这些故事意象无关，它照样是美的，但是，若能把琴曲放在特定的环境中，有了心里的准备而去欣赏，那么你所听到便更有深度了，这些深度的体验会让你真切地喜爱上了每一首琴曲。

      对于琴曲的体验有不同的时机和缘分。而随着时长，体验中可以既保留着初听时的感觉，也会不断地加入新鲜的感受。琴曲的故事，遥远而寥寥数语，其中的细节却要让弹琴人来添加，根据每个人的不同。但是，不同之处所弹奏出来的美，每一样都是可以被别人理解的，虽然不是全部的人。你会再一次感受到，人们的情感可以是相通的。

      昨天在市中心的车站内吃饭，对面两只肥大的鸽子一上一下地在做日常活动。转身，前进，都是三两下，跳得非常灵活，与它们硕大的身体相比，简直有点不可思议，却非常有趣，让看的人觉得很开心。对面的老妇也注意到了眼前的鸽子，用善意的目光观察了一会，但这样温柔的目光毕竟与那鸽子又圆又大的眼神交汇不到一处，也就把视线离开了。

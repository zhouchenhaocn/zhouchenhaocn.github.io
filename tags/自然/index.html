  • 跑步

    2014-04-24 |
    • { 自然 }
    • { 人文 }
    • { 随笔 }

      我喜欢上了家附近reserve的跑道，从第一次跑步到现在，脚下和眼前都熟悉起来，踩在路上，眼望前方，全身舒展开来，非常舒服。习惯了行进中有节奏的呼吸，听着脚下稳稳地声音，感觉可以一直跑下去。

      有风迎面持续吹来的时候，有阳光倾斜地照耀的时候，跑步变得更加享受。清爽的空气，傍晚的草坪，很多小小的鸟儿贴着草尖飞来飞去，近处观察，鸟儿油亮的白色腹部都看得一清二楚，滑翔而过，像在绿色的云朵上自由穿梭一样，想到这里，眼前的景色变得奇妙起来。

      但是，脚踩的大地总是比高处的云朵更加令人踏实和亲近，鸟儿们即使是一直在天空，大地想必还是它们的归宿。

      人们相逢相离，其中有断不了的想念和情愫，有些人彼此在向往着对方和对方的土地，有些人此时相隔千里。当这样的时空无论怎样都穿越不透的时候，那就怀着一颗宁静的心在大地上奔跑吧，大地藏着彼此的根，那是它的秘密。

  • 虫声

    2013-12-14 |
    • { 自然 }
    • { 随笔 }

      记录一下今天的好天气。

      入夏以来经历过暴升的高温，随后是连续的大风大雨，到最近天气才渐渐缓和。风清日宜，蓝天白云，空气中混合了花和草的味道，让人忍不住留恋在外。

      往常的高温天，白天是恼人的蝉声，晚上是遍地的虫鸣，温度越高，叫声越是团结肆意。在我的印象里，虫声就是为这样的夏天而鸣的。久居在城市的楼房里，在背景的喧闹中唯有如此嘹亮而持久的虫声才能被我注意到。这样的注意尚且是被动的，更不要说其他细腻的虫声了。书中的秋虫对我来说是个陌生的词汇，它们的鸣叫在特殊的季节里如摇铃般的悦耳，然而我全不曾注意，甚至怀疑起秋虫的存在了。

  • 永生

    2013-05-31 |
    • { 自然 }
    • { 随笔 }

      在写论文前，准备论文却是一段很快乐的时光。上星期终于提交了本学期的一篇重头，由第一堂实验课每个人领到一个胚胎切片开始，论文的准备实际上就开始了，这是三个月以前。

      Developmnental Biology I 和天文一样都很抽象，不同的是前者围绕着一个圆球，在这个圆球里，胚胎上演着各种各样的变化，每一部分都有着独自的归宿。直到最终长成子宫里即将出生的婴儿。

      学的时候真的是想破脑袋，像了解宇宙一样的去了解这一个小小的圆球，网上的视频很有用但不多，很多时候不得不借助各种角度的图片来理解某一个过程的动态。

      这门课并不是我的主修，但是很有趣。论文题目是每个人论述一种细胞种类，如干细胞，神经细胞，肌细胞，表皮细胞等。从其在胚胎中的最初分化，繁殖，迁移，到形成组织，器官，最终怎样在人体中发挥作用的过程。要论述的细胞连同其周围的环境还要在自己的胚胎染色切片上识别出来并拍照，引用在论文里。

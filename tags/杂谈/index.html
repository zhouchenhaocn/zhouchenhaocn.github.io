  • 写给自己的话

    2018-06-24 |
    • { 生活 }
    • { 杂谈 }

      有很多永恒的话题是跨越时代的，它们往往根植于我们的生活之中。不同的人为之建言献策，其中不乏有才华的人，他们善于精炼出一些细腻而博大的思想，这些思想并非遥不可及的概念，更不是唯一的真理，而是他们自己的人生思索，同时又是我们的共鸣。

      我爱这种共鸣。当你通过不同的方式，无论是在现实中还是艺术中，接近其他人的生活或者他们的思考方式时，你的大脑会帮助你进行“分析”，让你能够对对方进行判断。“判断”这个词其实过于狂妄了，我更愿意用“感同身受”这个过程。“分析”和“判断”往往预示着一个非好即坏的结果，但在“感同身受”中，你自己也参与其中，即便是那些丑陋而卑微的习惯，也可能在当下被理解。对方像一面镜子一样让你反观自己，而只有当你对自己也足够了解时，你才有可能更好地认识对方。在与人的相处中，我希望未来的自己能够慢慢摒弃“判断”的思维方式，而更多地用“感同身受”的过程来了解一个人。

  • 母亲节

    2018-05-12 |
    • { 生活 }
    • { 杂谈 }

      在高中的校园里觉得十八岁是最好的年纪，到了大学觉得二十岁是最好的年纪，而随着年纪再增加一点，又会觉得此刻才是最好的。和父亲母亲通电话，越来越感觉到他们作为“过来人”而拥有的一种睿智，并非世俗的，而是一种在历经生活的体悟之后满怀的一种善意。他们充满道理，温柔而又迫切地告诉你怎么更好地面对这个世界，告诉你他们的想法，为什么他们会这么想，渐渐把越来越多的东西给你看。你是他们眼中的孩子，他们会看着你，提醒你，给你一盏明灯，但同时你又已经是他们眼中的骄傲，他们把你作为一个与他们平等的对象看待，永远做你的后援，支持你。你看似离开他们，但是你却没有这样的感觉，除了亲情，还有一种你可以感觉到的力量在周围：你在同不可多得的人交谈。他们收藏着你将要经历的悲喜苦乐，他们温柔而又迫切地想要和你分享，而你也有同样的心去回复他们。

  • 琴会

    2014-10-03 |
    • { 生活 }
    • { 杂谈 }

      传说七弦琴在精通的人手中能够惊天地泣鬼神，各种音调转折能使星辰移动，风雷轰鸣，雨雪自积聚来的云中飞降大地。这真的是很浪漫。其实七弦琴本身并没有十足的力量，一张小小的琴，传出低调的琴声，时刻准备淹没在万物与世界之中。要说琴的奇妙力量，其实是心的力量。天地鬼神，雨雪星辰，大概只是在内心翻涌流动，但是，这就足够伟大了。

      我还是愿意保留对传说中精通古法的弹奏者的憧憬。那样的演奏，竟能使鬼神为之倾听而感动，更和自然的力量产生共鸣，好像世界和心之间真的相连一样。弹琴在此刻并非只是自我的事，而是倾注着弹奏者庄严和广阔的心，并非只是为谁弹奏，而是充盈着一种如斋戒和沐浴一般的对这个世界的崇敬。

  • 寄宿家庭的记忆

    2014-08-03 |
    • { 生活 }
    • { 杂谈 }

      今天出门，校区旁Monash College里陆陆续续出来了很多刚到这里的学生，推着机场下来还未开封的行李，每个人的身旁都有一位寄宿家庭的家长陪伴。家长们手里拿着学生的资料，显然是刚和学生碰面的。学生们从机场直接被送到这里，和等待他们的家庭欣喜见面，然后一起回家。

      阳光明媚的上午，我在车站等车，站在站台旁边树与树之间大片的阳光下，寒冷的感觉从身上渐渐消失。车站的方向正是Monash College的出口，我可以观察到他们走出来的情形，可以听到家长们对自己学生的问候和指示，更可以看到学生们的脸和身影：初到墨尔本，初到异地，和寄宿家庭碰面而初步扎根的他们有着那种很纯很干净的表情，像刚接触世界一样的羞涩却又勇往直前。他们将要开始在这里的学习和生活，全新全新的。

  • 虫声

    2013-12-14 |
    • { 生活 }
    • { 杂谈 }

      记录一下今天的好天气。

      入夏以来经历过暴升的高温，随后是连续的大风大雨，到最近天气才渐渐缓和。风清日宜，蓝天白云，空气中混合了花和草的味道，让人忍不住留恋在外。

      往常的高温天，白天是恼人的蝉声，晚上是遍地的虫鸣，温度越高，叫声越是团结肆意。在我的印象里，虫声就是为这样的夏天而鸣的。久居在城市的楼房里，在背景的喧闹中唯有如此嘹亮而持久的虫声才能被我注意到。这样的注意尚且是被动的，更不要说其他细腻的虫声了。书中的秋虫对我来说是个陌生的词汇，它们的鸣叫在特殊的季节里如摇铃般的悦耳，然而我全不曾注意，甚至怀疑起秋虫的存在了。

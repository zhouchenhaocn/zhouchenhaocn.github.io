  • 写给自己的话

    2018-06-24 |
    • { 生活 }
    • { 杂谈 }

      有很多永恒的话题是跨越时代的，它们往往根植于我们的生活之中。不同的人为之建言献策，其中不乏有才华的人，他们善于精炼出一些细腻而博大的思想，这些思想并非遥不可及的概念，更不是唯一的真理，而是他们自己的人生思索，同时又是我们的共鸣。

      我爱这种共鸣。当你通过不同的方式，无论是在现实中还是艺术中，接近其他人的生活或者他们的思考方式时，你的大脑会帮助你进行“分析”，让你能够对对方进行判断。“判断”这个词其实过于狂妄了，我更愿意用“感同身受”这个过程。“分析”和“判断”往往预示着一个非好即坏的结果，但在“感同身受”中，你自己也参与其中，即便是那些丑陋而卑微的习惯，也可能在当下被理解。对方像一面镜子一样让你反观自己，而只有当你对自己也足够了解时，你才有可能更好地认识对方。在与人的相处中，我希望未来的自己能够慢慢摒弃“判断”的思维方式，而更多地用“感同身受”的过程来了解一个人。

      包括自己，当代的很多年轻人与他们的父母辈，以及更长的长辈们相比，都缺少一种魄力，或者说是一种闯劲和韧性。我们似乎都没有意识到，这些本来应该受父母和长辈们耳濡目染而融入我们成长的东西，是何等的宝贵。时代的变化飞快，信息和交通的获取都无比便利。长大后的我们，和大家庭，包括和父母的距离都在渐渐变大。我们每天都在接触庞大的信息，似乎可以从中学到更多。我们忽略了家庭的作用，其实家庭的教育对一个已经成长了的孩子来说依旧是延续着的。一个孩子往往在家庭中充满着在其他地方很难拥有的天生的感受力：对于生命的情感，人与人的关系，善良，苦难，悲和喜。这些最本质的东西，一个孩子会在他的家庭中获得最初也是最强烈的启蒙，并且这样的启蒙在孩子人生的不同阶段还会延续下去。他会主动去体会父母和长辈的的辛劳，以及他们人生的精神面貌，这些都是毫无遮掩的。这些真实的东西会帮助孩子解答他们自己的困惑，帮助他们不再被看似庞大的世界，多元的社会，激烈的竞争和每天纷杂的信息所吓倒，从而活出自己的人生，并且更重要的是，保持一颗善良的本心。

      我的骨子里其实最喜欢的是一种平民文化。它并非平庸，而是一种深刻的，内敛而有韧性的精神。它鼓励人们向上，同时提供一种与人生温柔和解的方式。我不知道怎样更确切地描述它，但我想它是否就是禅。最深刻的东西往往植根于平民，也就是我们每一个人的身上心里，它承载着我们情感中最真实而精炼的部分，并且会永恒地传承下去。

      在写文章的时候我感觉四散在周围的注意力都回来了，并非强迫的，而是一种天然的过程。博客里藏着一个沉静而充盈的气场，这源自我和时间在做的长久的游戏。有很多永恒的话题，十年之前我想着，十年后我还在想。

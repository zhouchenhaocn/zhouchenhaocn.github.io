  • 芭蕾舞《灰姑娘》

    2018-09-08 |
    • { 人文 }
    • { 随笔 }

      人生第一次在现场看芭蕾舞剧，十分难忘，观看的时候脑海里有很多新鲜的体会和有意思的想法，在幕间和演出结束后趁着思想活跃赶紧记了下来，现在回家了，准备安安静静地再做一些延伸。其实很多感受还是无法用言语轻易描述，需要自己继续回味，尽管如此，自己还是要把大体的感受记录下来，作为以后的参考。

      芭蕾舞剧的演出舞台底下会坐着一个完整的乐团，虽然比不上大型交响乐团的规模，但指挥，管弦和打击乐器的部分应有尽有。看舞的时候，耳边可以清晰地分辨出各种乐器的不同音色，它们此起彼伏，互相配合，作为一个整体让音乐持续流动。以前看视频的时候觉得古典音乐和芭蕾舞是绝配，今天加深了印象。真的很奇妙，芭蕾就像是打开古典音乐的一把钥匙。

      一场舞剧不仅仅是舞蹈。坐在剧院里的时候，我可以明白来自不同领域的人的共同的付出。从确定一个故事开始，有人编曲，有人编剧，有人编舞，还有人负责服装，舞台的布景，灯光，最后是演员和乐团，缺一不可，并且他们之间需要恰到好处的合作。我很惊讶，并且由衷地佩服，今天展现在舞台上的每一部分都让人享受。作为一名细心的观众，除了陶醉于对芭蕾舞本身那种脱俗的魅力外，还会在意每一个演员服装的配色，以及当演员做出动作时，他们的服装因为材质，厚度等等方面的设计而形成的特别的飘扬和摆动。当灯光照在演员的配饰上那不时闪烁的情形，同样会抓住观众的眼睛。当春，夏，秋，冬四位精灵出场时，他们每一个人的衣服的选色，以及起舞时的背景都是经过精心准备的，那种审美的契合给观众带来的心灵愉悦让人无法用语言形容。观众内心会明白这些精彩的地方来自每个在舞台后的人的创造与付出，他们慢慢地打磨，最终成就了一件成熟的艺术作品。

  • 写给自己的话

    2018-06-24 |
    • { 人文 }
    • { 随笔 }

      有很多永恒的话题是跨越时代的，它们往往根植于我们的生活之中。不同的人为之建言献策，其中不乏有才华的人，他们善于精炼出一些细腻而博大的思想，这些思想并非遥不可及的概念，更不是唯一的真理，而是他们自己的人生思索，同时又是我们的共鸣。

      我爱这种共鸣。当你通过不同的方式，无论是在现实中还是艺术中，接近其他人的生活或者他们的思考方式时，你的大脑会帮助你进行“分析”，让你能够对对方进行判断。“判断”这个词其实过于狂妄了，我更愿意用“感同身受”这个过程。“分析”和“判断”往往预示着一个非好即坏的结果，但在“感同身受”中，你自己也参与其中，即便是那些丑陋而卑微的习惯，也可能在当下被理解。对方像一面镜子一样让你反观自己，而只有当你对自己也足够了解时，你才有可能更好地认识对方。在与人的相处中，我希望未来的自己能够慢慢摒弃“判断”的思维方式，而更多地用“感同身受”的过程来了解一个人。

  • 母亲节

    2018-05-12 |
    • { 人文 }
    • { 随笔 }

      在高中的校园里觉得十八岁是最好的年纪，到了大学觉得二十岁是最好的年纪，而随着年纪再增加一点，又会觉得此刻才是最好的。和父亲母亲通电话，越来越感觉到他们作为“过来人”而拥有的一种睿智，并非世俗的，而是一种在历经生活的体悟之后满怀的一种善意。他们充满道理，温柔而又迫切地告诉你怎么更好地面对这个世界，告诉你他们的想法，为什么他们会这么想，渐渐把越来越多的东西给你看。你是他们眼中的孩子，他们会看着你，提醒你，给你一盏明灯，但同时你又已经是他们眼中的骄傲，他们把你作为一个与他们平等的对象看待，永远做你的后援，支持你。你看似离开他们，但是你却没有这样的感觉，除了亲情，还有一种你可以感觉到的力量在周围：你在同不可多得的人交谈。他们收藏着你将要经历的悲喜苦乐，他们温柔而又迫切地想要和你分享，而你也有同样的心去回复他们。
